<details>
<summary><b>🔷 EchoGuard / SentinelAI — Behavioral Authentication Platform (MNNIT Cybersecurity Hackathon 2026)</b></summary>
<br>

> A passive continuous authentication system using multi-modal behavioral biometrics — keystroke dynamics, gyroscope telemetry, and touch pressure — to detect session hijacking and unauthorized access in real time.

| Attribute | Detail |
|:--|:--|
| **Stack** | React Native · FastAPI · Python · Isolation Forest · Sensor Fusion |
| **Scale** | Continuous per-session profiling; real-time anomaly scoring |
| **Performance** | Sub-second behavioral deviation detection; lightweight mobile sensor pipeline |
| **Security** | Passive authentication; no user friction; adversarial resistance by design |
| **Impact** | MNNIT Cybersecurity Hackathon 2026; novel biometric layer for mobile banking |
| **Repository** | [View on GitHub →](https://github.com/Sakshya10027/echoguard) |

EchoGuard represents a new paradigm in mobile authentication: instead of one-time login verification, the system continuously profiles the user's behavioral fingerprint throughout a session. Gyroscope telemetry captures how a user holds their device; keystroke dynamics encode typing rhythm; multi-touch pressure signatures create a composite behavioral model. Isolation Forest detects deviations from the established profile in real time, triggering re-authentication or session termination when anomalies exceed threshold — all with zero user friction under normal conditions.

</details>
